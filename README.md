# 📡 Telco Churn AI  
### Modern Single-Page Customer Churn Intelligence Dashboard  

An AI-powered web application that predicts customer churn risk using an XGBoost machine learning model and presents results through a modern SaaS-style Streamlit dashboard.
## 🚀 Features

- 🎯 Real-time churn prediction  
- 📊 Interactive probability gauge (Plotly)  
- 💡 Automated retention strategy suggestions  
- 📁 Bulk CSV batch prediction  
- ✨ Demo data loader  
- 🧹 Clear input functionality  
- 🌌 Modern glassmorphism UI  
- ☁️ Render deployment compatible

## 🧠 Model Overview

This system uses:
- **XGBoost Classifier**
- Trained on the Telco Customer Churn dataset
- One-hot encoded categorical features
- Feature-aligned deployment architecture

The model outputs:
- Binary churn prediction (0 = Stay, 1 = Churn)
- Probability score (0–100%)
